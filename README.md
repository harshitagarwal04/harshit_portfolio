# Developer Portfolio - Harshit Agarwal

A minimalist, terminal-inspired, backend-focused developer portfolio built with Next.js, Framer Motion, and PixiJS. 

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** TailwindCSS 
- **Animations:** Framer Motion
- **Visuals:** PixiJS (Interactive node background)
- **Icons:** Lucide React

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure
- `/app` - Next.js App Router (Layout & main page integration)
- `/components/sections` - Core portfolio blocks (Hero, About, Experience, CaseStudy, etc.)
- `/components/ui` - Primitive building blocks (Terminal windows, Wrappers)
- `/components/pixi` - Lightweight canvas node background rendering

## Deployment
This project is configured to deploy out-of-the-box perfectly on [Vercel](https://vercel.com/new).
